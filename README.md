# testWeb

供应商报价管理系统前端项目

## 项目简介

这是一个基于 Vue.js 开发的供应商报价管理系统前端应用。

## 功能特性

- 供应商登录
- 报价列表管理
- 报价详情查看
- 报价历史记录
- 倒计时功能

## 在线访问

GitHub Pages: [https://wuliwm.github.io/testWeb/](https://wuliwm.github.io/testWeb/)

## 技术栈

- Vue.js
- Vite
- Element Plus

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 部署

项目使用 GitHub Pages 进行自动部署。
